DEPO ORM - AI-Powered Online Reputation Management (2026 Edition)
DEPO ORM is a cutting-edge reputation management platform designed for the hospitality and service industries. It leverages the power of Artificial Intelligence to automate the tracking, analysis, and response generation for customer reviews, ensuring a consistent and professional brand voice.

🌟 Key Features
Smart Review Fetching (DP-01 Compliance):

Users can input a hotel name into the search bar.

The system queries the database to retrieve the 5 most recent reviews associated with that specific property.

AI-Driven Drafting:

Integrated with Gemini 3.5 Flash (the latest 2026 AI model).

Automatically generates 3 distinct response styles: Standard, Friendly, and Crisis Management.

Real-time Status Management:

Seamlessly connected to Supabase to update review statuses from Pending to Resolved upon response approval.

Modern Dashboard UI:

Built with Next.js 16 for a high-performance, responsive, and intuitive user experience.

🛠 Tech Stack
Frontend: Next.js 16 (App Router), TypeScript, Tailwind CSS.

AI Engine: Google Gemini 3.5 Flash API (JSON Native Mode).

Backend & Database: Supabase (PostgreSQL) with Real-time capabilities.

Infrastructure: Deployed on Vercel with automated GitHub CI/CD.

🚀 Local Installation & Setup
Clone the repository:

Bash
git clone https://github.com/hanhgcd220331-eng/ai-orm.git
cd depo-orm-app
Install dependencies:

Bash
npm install
Configure Environment Variables (.env.local):
Create a .env.local file in the root directory and add your credentials:

Code
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
GEMINI_API_KEY=your_gemini_api_key
Run the development server:

Bash
npm run dev
📊 Database Schema (SQL)
Run the following script in your Supabase SQL Editor to initialize the database:
SQL
CREATE TABLE reviews (
  id int8 GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  customer_name text DEFAULT 'Anonymous Customer',
  content text NOT NULL,                                -- Review content containing the hotel name
  ai_response text,                                     -- The approved AI-generated response
  status text DEFAULT 'Pending',                        -- Processing status (Pending/Resolved)
  created_at timestamp with time zone DEFAULT now()     -- Timestamp for "Latest 5" logic
);
🛡 Security & Best Practices
Zero-Footprint Security: All API Keys are managed via Vercel Environment Variables; no sensitive data is hardcoded or exposed in the public repository.

Performance Optimization: Implemented ilike queries and limit(5) constraints to ensure rapid data retrieval and minimal server load.

Git Compliance: Strict .gitignore configuration to prevent accidental leakage of local configuration files.
